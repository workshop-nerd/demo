Progetti Finalisti

Galleria dell'Innovazione

Nove progetti che dimostrano come le giovani donne italiane stiano guidando la trasformazione digitale.

🧠
MediAI — Diagnosi Predittiva Oncologica

Modello di deep learning per il rilevamento precoce di tumori da immagini radiologiche, con accuratezza superiore al 94%.

TensorFlow Python DICOM CNN

Team: Giulia Rossi, Università di Milano

🧠
LinguAI — Inclusione Linguistica

Sistema NLP per la traduzione automatica in Linguaggio Facile da Leggere (LIS), abbattendo le barriere per persone con disabilità cognitive.

NLP BERT FastAPI React

Team: Elena Moretti, Università Federico II

🧠
CropMind — Agricoltura di Precisione

Piattaforma AI che analizza dati satellitari e sensori IoT per ottimizzare irrigazione, fertilizzazione e raccolta nelle piccole aziende agricole.

Satellite API IoT PyTorch GIS

Team: Martina Bianchi, Università di Bologna

🌿
CarbonTrack — Impronta Digitale Zero

App mobile che misura e riduce l'impronta di carbonio personale attraverso gamification, suggerimenti personalizzati e challenge di comunità.

Flutter Firebase Carbon API UX Design

Team: Sofia Ferrari, Politecnico di Torino

🌿
AquaSense — Gestione Idrica Intelligente

Rete di sensori low-cost e dashboard open-source per il monitoraggio dei consumi idrici nei condomini, con risparmio medio del 30%.

Arduino MQTT Grafana Node.js

Team: Valentina Conti, Università di Firenze

🌿
WasteNet — Economia Circolare Digitale

Marketplace B2B che connette aziende produttrici di scarti industriali con imprese che li trasformano in nuovi materiali, riducendo il conferimento in discarica.

Vue.js PostgreSQL REST API Docker

Team: Alessia Greco, Università di Bari

🤝
SafeHome — Contrasto alla Violenza Domestica

App discreta con funzione SOS nascosta e connessione diretta ai centri antiviolenza, sviluppata in collaborazione con associazioni del terzo settore.

React Native Geolocation Encryption NGO API

Team: Chiara Romano, La Sapienza Roma

🤝
BridgeWork — Inclusione Lavorativa

Piattaforma di matching intelligente tra persone con disabilità e aziende inclusive, con percorsi di formazione personalizzati e tutoraggio digitale.

Next.js ML Matching OAuth 2.0 Accessibilità

Team: Laura Esposito, Università di Padova

🤝
EduBridge — Scuola Digitale Equa

Piattaforma e-learning offline-first per studenti in aree remote o con connessione instabile, con contenuti adattivi e supporto multilingue.

PWA IndexedDB i18n Svelte

Team: Sara Lombardi, Università di Palermo
